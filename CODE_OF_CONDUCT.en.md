# Code of Conduct

English | [中文](CODE_OF_CONDUCT.md)

`vexra-ldb` aims to maintain a professional, friendly, and problem-focused collaboration environment. Please follow these expectations when participating in issues, PRs, code reviews, documentation discussions, or other project communication.

## Encouraged Behavior

- Respect participants with different backgrounds and levels of experience.
- Focus on facts, code, tests, and design constraints.
- Provide reproducible evidence for risks and disagreements.
- Suggest improvements clearly and constructively.
- Acknowledge mistakes and correct them promptly.

## Unacceptable Behavior

- Personal attacks, harassment, discriminatory language, or threats.
- Exposing someone else's private information or disclosing security issue details without authorization.
- Repeatedly posting irrelevant content in technical discussions.
- Intentionally encouraging others to run operations known to damage data or security.

## Enforcement

Maintainers may respond to violations by reminding participants, editing or hiding content, closing discussions, limiting participation, or reporting behavior to the platform. For security issues, follow the private reporting process in `SECURITY.md` first.
